Структура проекта:

├── bower_components - Необходимые компоненты для работы
├── build - Скомпилированные файлы посредством gulp
├── macket_layout - Исходный макет проекта
├── src - Исходные файлы (sass, fonts, images)
├── Gulpfile.js - Gulp-файл, настроек для компиляции
├── index.html - Главная страница
└── *.json - Файлы настроек для сборки проекта

Используемые компоненты:
 - jQuery
 - Semantic-UI
